## 1. 스마트정수장 구축 및 운영 프로젝트

### 프로젝트 개요
착수 공정 제어 과정에서 반복적으로 발생하던 정수지 수위 이탈 문제를 개선하기 위한 운영 최적화 프로젝트

### 주요 역할
- SCADA 시계열 데이터 분석
- 운영 로그 기반 문제 패턴 분석
- 예측 기반 제어 구조 개선
- 현장 운영 환경 반영 제어 로직 개선
- 현장 근무자 협업 및 운영 안정화 지원

### 문제 상황
기존 유입 유량 예측 방식으로는 특정 시간대 운영 변동성을 충분히 반영하지 못해 정수지 수위 이탈 문제가 반복 발생

### 개선 내용
- 단순 유입 유량 예측 방식 → 정수지 수위 예측 기반 구조로 전환
- 운영 흐름 및 현장 조건 반영 알고리즘 개선
- 반복 제어 오류 감소 및 운영 안정화 개선

### 사용 기술
`Python` `Pandas` `Time Series Data` `SCADA` `DB`
